# Retail Sales Analysis

## Project Overview
This project analyzes retail sales data using Python, Pandas, and Matplotlib.

## Dataset
- online_retail.csv

## Data Cleaning
- Removed duplicate records
- Removed rows with missing Description values
- Converted InvoiceDate to datetime format
- Created Total_Sales column
- Extracted Month and Year from InvoiceDate

## Analysis Performed
- Top 10 Products by Revenue
- Monthly Sales Trend
- Top 10 Countries by Revenue

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Jupyter Notebook

## Files
- Retail_Sales_Analysis_Preparation.ipynb
- cleaned_retail_sales.csv
- online_retail.csv
